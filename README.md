# You_Tube_Clone-Frontend
This repository contains the frontend code for a YouTube clone project, implemented using HTML and CSS only. The project replicates the essential visual elements of the popular video-sharing platform, YouTube, focusing on frontend development without any JavaScript or backend functionalities.
# Futures
- The frontend ensures a responsive layout for optimal viewing on various devices.
- Displays video thumbnails with relevant information.
- Provides a user-friendly interface for browsing videos.
- The project includes static content and does not involve dynamic data fetching.
# Technologics Used
HTML, CSS
